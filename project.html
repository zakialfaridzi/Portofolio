MyVoQu (My Voice of Alquran)

A web-based Alquran learning application with social media concepts such as posts, likes, and comments, built with CodeIgniter, Jquery, and Bootstrap.
This Alquran learning system lets you learn Alquran interactively with its social media concepts and live video chatting. We also have learning groups so that users can learn in a not very crowded ecosystem.
